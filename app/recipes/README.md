# Recipes

See the full documentation at [docs/recipes.md](../../docs/recipes.md).

## Quick Start

1. Create a folder under `app/recipes/screens/`
2. Add your component (e.g., `my-recipe.tsx`) and optional `getData.ts`
3. Register it in `app/recipes/screens.json`

## Structure

```
app/recipes/screens/
├── my-recipe/
│   ├── my-recipe.tsx    # React component
│   └── getData.ts       # Optional data fetcher
```

Visit `/recipes` in the app to preview your screens.
